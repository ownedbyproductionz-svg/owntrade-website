What ownTRADE does
Everything in one app.

From strategy development to automated execution — ownTRADE covers the complete workflow. No cloud dependency, no hidden fees.

Scalping Bot v3.8

Fully automated trading on Binance Futures. Real fill prices, ATR-based SL/TP, trailing stop and score-based signal system with 11 indicators.

Backtester

50,000+ real orderbook snapshots. Automatic SL/TP optimizer and greedy weight optimizer. Transfer the best values to the bot with one click.

Live Dashboard

Real-time overview of PnL, open positions, trades and bot status. Start and stop the bot directly from the Electron desktop app.

Fully Configurable

Position size, leverage, ATR multipliers, indicator weights, min-score, cooldowns — all adjustable via sidebar, transferable directly to the bot.

Self-Hosted

Runs entirely on your own machine. Your API keys, your data, your control. No external servers required for core functionality.

Electron Desktop App

Netflix-style launcher, dedicated window per tool, seamless IPC communication between UI and Python backend. Ships as a single EXE.
